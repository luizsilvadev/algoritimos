Exercícios

1) Identifique o(s) problema(s) com o algoritmo de busca sequencial abaixo e realize os ajustes para que ele funcione corretamente

int buscaSeq (int x, int n, int v[]) {
   int j = 0;
   while (v[j] < x && j < n) ++j;
   return j; 
}

2) O código abaixo deve realizar a busca binária em um arranjo A de tamanho N. Entretanto, para compreender melhor os passos e comparações realizadas, modifique o código para que, a cada nova iteração, o número do meio e a porção a ser analisada sejam impressos na saída padrão (monitor).

    int binaria (int A[], int i, int f, int b){
        int m;
        while (i <= f){
            m = (i+f)/2;
            if (A[m] == b)
                return m;
            else{
                if (A[m] > b)
                    f = m-1;
                else 
                    i = m+1;
            }        
        }
        return -1;
    }
